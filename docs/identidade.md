Este documento apresenta uma proposta de identidade visual para o sistema "Assembleia Digital", com o objetivo de conferir um aspecto profissional, sofisticado e tecnológico, sem excessos. A diferenciação visual dos perfis de usuário será um ponto chave, utilizando cores distintas para facilitar a identificação e a usabilidade.

## 2. Conceito e Estilo

O conceito central busca transmitir **confiança, modernidade e eficiência**. A estética será **minimalista e limpa**, com foco na legibilidade e na experiência do usuário. Elementos tecnológicos serão incorporados de forma sutil, através de tipografia, ícones e um uso estratégico de cores e gradientes.

## 3. Paleta de Cores

A paleta de cores é dividida em cores primárias (para a marca e elementos de destaque), cores neutras (para fundos, textos e bordas) e cores de perfil (para diferenciar os tipos de usuário).

### 3.1. Cores Primárias e de Destaque

Estas cores serão utilizadas para elementos de branding, botões de ação principal, gráficos e acentos visuais que necessitam de destaque.

| Nome da Cor | Código Hexadecimal | Descrição                                                              |
| :---------- | :----------------- | :--------------------------------------------------------------------- |
| Azul Profundo | `#2C3E50`          | Principal cor da marca, transmite seriedade e tecnologia.              |
| Ciano Elétrico | `#00BCD4`          | Cor de destaque, para interatividade e elementos ativos.               |

### 3.2. Cores Neutras

Essas cores formam a base do design, garantindo legibilidade e um visual limpo.

| Nome da Cor | Código Hexadecimal | Descrição                                                              |
| :---------- | :----------------- | :--------------------------------------------------------------------- |
| Branco Puro | `#FFFFFF`          | Fundos de cartões, áreas de conteúdo.                                  |
| Cinza Claro | `#F8F9FA`          | Fundos de seções, áreas secundárias.                                   |
| Cinza Médio | `#CED4DA`          | Bordas, divisores, ícones inativos.                                    |
| Cinza Escuro | `#495057`          | Textos principais, títulos.                                            |
| Preto Suave | `#212529`          | Textos de alto contraste, cabeçalhos.                                  |

### 3.3. Cores dos Perfis de Usuário

Cada perfil terá uma cor distintiva para facilitar a identificação visual rápida em listagens, dashboards e outras interfaces. As cores foram escolhidas para serem distintas, mas harmoniosas com a paleta geral.

| Perfil             | Código Hexadecimal | Descrição                                                              |
| :----------------- | :----------------- | :--------------------------------------------------------------------- |
| **Administrador**  | `#E74C3C`          | **Vermelho Vibrante:** Transmite autoridade e atenção, ideal para o controle administrativo. |
| **Responsável**    | `#28A745`          | **Verde Sucesso:** Sugere responsabilidade, aprovação e segurança.     |
| **Máquina de Votação** | `#FFC107`          | **Amarelo Alerta:** Indica um estado de prontidão e importância funcional. |
| **Mesário**        | `#6F42C1`          | **Roxo Dignidade:** Representa a imparcialidade e a formalidade do papel. |

## 4. Tipografia

Para manter a estética profissional e tecnológica, sugere-se o uso de uma família de fontes moderna e legível.

*   **Fonte Principal (Títulos e Destaques):** `Montserrat` ou `Open Sans` (Sans-serif, geométrica, moderna e limpa).
*   **Fonte Secundária (Corpo de Texto):** `Roboto` ou `Lato` (Sans-serif, alta legibilidade em diversos tamanhos).

## 5. Elementos de Interface

*   **Botões:** Estilo "flat" ou "ghost" com cantos levemente arredondados. Cores primárias para ações principais, neutras para secundárias.
*   **Cartões/Painéis:** Fundos brancos ou cinza claro, com sombras sutis para profundidade (box-shadow: `0 2px 4px rgba(0,0,0,.05)`).
*   **Ícones:** Estilo "line-art" ou "preenchido" minimalista, preferencialmente em tons de cinza ou ciano elétrico para interatividade.
*   **Navegação:** Cabeçalho escuro (Azul Profundo ou Preto Suave) com texto e ícones em branco ou ciano elétrico para contraste.

## 6. Considerações de Acessibilidade

As combinações de cores propostas foram pensadas para garantir um contraste adequado, facilitando a leitura e a diferenciação para usuários com diversas necessidades visuais. Recomenda-se testar as combinações de cores para garantir conformidade com as diretrizes WCAG (Web Content Accessibility Guidelines).

## 7. Conclusão

Esta proposta visa estabelecer uma base sólida para a identidade visual do "Assembleia Digital", combinando profissionalismo, sofisticação e um toque tecnológico. A aplicação consistente destas diretrizes garantirá uma experiência de usuário coesa e visualmente agradável.
